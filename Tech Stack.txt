
Tech Stack:
1.	HuggingFace
→ LLM and Embedding Model
(Free, Mistral AI used as LLM)
2.	FAISS
→ Local Vector Store for storing your embeddings
(Meta / Facebook AI Similarity Search)
3.	LangChain
→ Generative AI framework to interact with LLM
4.	PyPDF
→ Library for reading contents of a PDF file
5.	Flask
→ To build backend APIs for handling user requests
6.	HTML / CSS
→ To make UI or frontend of the app
7.	Docker
→ For containerization of the app during deployment
8.	Aqua Trivy
→ For scanning Docker images for security issues and vulnerabilities
9.	Jenkins
→ For making CI/CD pipelines
10.	AWS Runner
→ To deploy and run your app on cloud with ease and reliability
→ It’s a service offered by AWS
11.	GitHub
→ It will work as a SCM for your project

Security Scanning with Trivy (Docker Image):
•	Trivy → Security Scanner (Docker Image)
•	Detects vulnerabilities
•	Finds weaknesses / flaws in Docker containers
•	These flaws can be exploited by hackers for unauthorized access
Example:
•	App → Flask = 2.0.1
→ Security bugs found
•	Flask ≤ 2.1.0
→ No security bugs
•	Trivy output:
o	Warning message → security bugs
o	Suggests upgrading Flask (2.0.1 → 2.1.0)
